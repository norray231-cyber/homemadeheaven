# Handmade Haven Project Guidelines

## Brand Identity
- **Brand**: Handmade Haven
- **Business**: Homemade Baked Goods & Savory Snacks
- **Audience**: Food lovers, families, party planners, students
- **Feel**: Fresh, homemade, warm, delicious, affordable

## Menu Categories
- Cookies (Classic, Chocolate Filled, Red Velvet, Monster)
- Brownies (Chococo, D'Choco, Oreo, Brownie Slab)
- Cupcakes (BananChoco, Choco Lava, Oreo Bliss, Double Choco, KitKat, Hazel Bliss, Marble Cake)
- Savory (Grillwich, FajGrill, Stuff'd Roll, Chicken Bread, Bread Kachori, Pops, Dynamite Chicken, Samosa, Zingy Parcel)

## Pricing (RS)
- Cookies: 100-150
- Brownies: 180-1000
- Cupcakes: 120-500
- Savory: 40-500

## Code Standards
- Maximum 300 lines per file
- Mobile-first responsive design
- Local assets only

## Color Palette
- Primary: #8B5A5A
- Secondary: #D4A574
- Background: #FDF8F5
